## *由于字库对简体中文适配程度较低，目前仅推荐使用繁体中文版*

---

使用方法：
1. 选择zh_CN.txt（简体中文）或zh_TW.txt（繁体中文）
2. 确保已完全关闭Interlude
3. 在Interlude-win64目录创建Locale文件夹
4. 以简体中文为例，将zh_CN.txt移动到Interlude-win64\Locale文件夹
5. 点开Interlude-win64\Data\options.json
6. 按Ctrl + F搜索language找到"Language": "en_GB",
7. 将en_GB替换为zh_CN
8. 检查改完后，该行是否为"Language": "zh_CN",
9. 点开游戏即可

---

使用方法：
1. 選擇zh_CN.txt（簡體中文）或zh_TW.txt（繁體中文）
2. 確認已完全關閉Interlude
3. 在Interlude-win64目錄建立Locale資料夾
4. 以繁體中文為例，將zh_TW.txt移動到Interlude-win64\Locale資料夾
5. 開啟Interlude-win64\Data\options.json
6. 按下Ctrl+F搜尋language，找到"Language": "en_GB",
7. 將en_GB替換為zh_TW
8. 檢查修改後該行是否為"Language": "zh_TW",
9. 開啟遊戲即可

---

注：
zh_TW(original)虽然更加符合繁体中文用户的语言习惯，但未对不能使用的字进行更改，会有显示bug，在unusablecharacter.txt有记录
