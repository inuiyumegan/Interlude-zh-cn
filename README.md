# English

## *Due to limited font support for Simplified Chinese, the Traditional Chinese version is currently recommended.*

### How to Use:

1. Choose szh_CN.txt (Simplified Chinese) or tzh_TW.txt (Traditional Chinese).
2. Make sure Interlude is completely closed.
3. Create a Locale folder inside the `Interlude-win64` directory.
4. Using Simplified Chinese as an example, move `szh_CN.txt` into the `Interlude-win64\Locale` folder.
5. Open `Interlude-win64\Data\options.json`.
6. Press Ctrl + F and search for `language`, then locate `"Language": "en_GB",`.
7. Replace `en_GB` with `szh_CN`.
8. After editing, make sure the line reads `"Language": "szh_CN",`.
9. Launch the game.
